# Anime Registration Platform

A professional anime registration platform with user authentication, admin panel, and Telegram bot integration.

## Features

- ✅ User Authentication (Login/Sign Up)
- ✅ Anime Registration Form with Video Upload
- ✅ Admin Panel with Management Dashboard
- ✅ Telegram Bot Integration
- ✅ Multi-language Support (English, Somali, Arabic)
- ✅ Dark Mode
- ✅ Responsive Design (Mobile-First)
- ✅ Gold & Soft Black Theme
- ✅ Secure Password Hashing
- ✅ Session Management
- ✅ Video Permanent Storage

## Tech Stack

**Frontend:**
- HTML5
- CSS3
- JavaScript (Vanilla)

**Backend:**
- Flask (Python)
- PostgreSQL

**Storage:**
- Permanent Video Storage System

**Integration:**
- Telegram Bot API

### Backend Setup
1. Create Python virtual environment
2. Install dependencies: `pip install -r requirements.txt`
3. Configure database
4. Run migrations
5. Start Flask server

### Frontend Setup
1. Open `index.html` in browser or serve with live server
